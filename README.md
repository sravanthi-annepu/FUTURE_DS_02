# 📊 Telecom Customer Churn Analysis

## 📌 Overview
This project analyzes customer churn behavior using Power BI to identify key factors affecting customer retention.

## 🛠 Tools Used
- Power BI
- Data Analysis
- Data Visualization

## 📊 Key Insights
- Month-to-month contracts have the highest churn
- Customers churn mostly in early tenure (0–5 months)
- Higher monthly charges increase churn risk
- Electronic check users show higher churn rates

## 💡 Recommendations
- Promote long-term subscription plans
- Improve onboarding experience
- Offer targeted retention incentives
- Encourage automatic payment methods

## 🚀 Outcome
This project demonstrates how data-driven insights can help improve customer retention in subscription-based businesses.
